## What changed in the modern platform

FreeSense is not just a renamed firewall image. The current line introduces a smaller, explicit
package contract; Bootstrap 5 package interfaces; split System and optional-package builds; atomic
Suricata configuration; and CI checks that connect package metadata to the actual build list.

<div class="docs-feature-grid">
  <div><b>Package clarity</b><span>Capabilities, services, support status, resource impact, and direct management links.</span></div>
  <div><b>Security integration</b><span>Suricata 8 events and validation, CrowdSec firewall decisions, and consent-driven ZeroTier routes.</span></div>
  <div><b>Release integrity</b><span>Every artifact records exact source, ports, OS-definition, and FreeBSD revisions.</span></div>
</div>

:::danger[Development is experimental]
Stable 1.0.x is the supported production line. Development 1.1 is experimental, intended only for
testing, and receives no support. The download page reads both independent release documents live.
:::
